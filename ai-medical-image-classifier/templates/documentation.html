üìä Dataset Information

Kaggle Chest X-ray Pneumonia Dataset

Source: Kaggle Dataset

Size: ~1.1GB compressed | Total Images: 5,840 X-rays

Training Set (5,216 images)

  • NORMAL: 1,341 images
  • PNEUMONIA: 3,875 images

Testing Set (624 images)

  • NORMAL: 234 images
  • PNEUMONIA: 390 images

Class Distribution: ~74% PNEUMONIA, ~26% NORMAL

This imbalance reflects real-world medical scenarios where pneumonia cases are more common.

Medical Significance

High recall for PNEUMONIA (96.67%) is crucial - it's better to have false positives that doctors can verify than to miss pneumonia cases that need immediate treatment.
